# 🚀 Ultra PC - UPS System

Système complet de gestion des commandes et de suivi de production développé pour Ultra PC.

## 📌 Description
UPS (Ultra PC System) est une application web de gestion industrielle permettant de suivre tout le cycle d’une commande : de la réception jusqu’à la livraison finale.

Le système est conçu pour optimiser la communication entre les différents services et automatiser le flux de production.

## ⚙️ Fonctionnalités

- 🔐 Authentification avec rôles (Admin, Réceptionniste, Gestionnaire, Assembleur, Emballeur, Expéditeur, Client)
- 📦 Gestion complète des commandes
- 🏭 Suivi de production en 8 étapes
- 📊 Gestion de stock des composants
- 🔔 Notifications en temps réel (WebSocket)
- 📱 Interface moderne responsive (mobile & desktop)
- 🧾 Historique et tracking des commandes

## 🧠 Stack technique

- Frontend : HTML / CSS / JavaScript (ou React)
- Backend : Node.js / Express (ou autre selon ton projet)
- Base de données : PostgreSQL
- Auth : JWT
- Temps réel : WebSocket

## 🎯 Objectif
Améliorer la productivité et le suivi interne dans une entreprise de fabrication informatique en centralisant toutes les opérations.
